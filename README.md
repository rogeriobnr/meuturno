Descrição do Projeto – Meu Turno

> Meu Turno é um sistema web PWA (Progressive Web App) desenvolvido para controle de turnos, geração de relatórios de manutenção, abertura de O.S. e gestão operacional de campo. O app funciona 100% offline, utilizando armazenamento local via LocalStorage, com backup e restauração em arquivos JSON.



O sistema foi projetado com foco em praticidade, agilidade e usabilidade em ambientes de difícil acesso à internet, como fazendas, oficinas móveis e áreas florestais.

Principais Funcionalidades:

Abertura de turnos com cadastro de frente, fazenda, equipe e frotas.

Geração e emissão de O.S. (Ordem de Serviço) com prévia em tela, compartilhamento por WhatsApp e histórico automático.

Relatórios de manutenção com cálculo automático de %DM (Disponibilidade de Máquinas), controle de horas planejadas e horas paradas, inclusive com sobreposição de horários.

Histórico completo com controle de status da O.S., edição, backup e restauração.

Backup completo de dados locais e importação/exportação em formato JSON.

Interface responsiva, leve, moderna e otimizada para uso em campo (tablet, celular ou desktop).

Navegação intuitiva por abas e uso de pop-ups para confirmação e seleção de dados.


Tecnologias utilizadas:

HTML, CSS e JavaScript (puro, sem frameworks)

Armazenamento em LocalStorage

PWA com Service Worker e Manifest JSON, permitindo uso offline com instalação no dispositivo.

Design focado em experiência de campo, com interface leve, limpa e objetiva.


Status do Projeto:

Implementação 100% funcional e finalizada com LocalStorage.

Pronto para uso em operações individuais ou pequenas equipes.

Arquitetura preparada para futura migração para IndexedDB, bancos externos ou expansão para multiusuário, caso necessário.
